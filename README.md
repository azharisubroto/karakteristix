## Getting Started

Install depedencies
```bash
npm install
#or
yarn install
```

First, run the development server:

```bash
npm run dev
# or
yarn dev
```

.env config example:
```env
SITENAME=KARAKTERISTIX
API_URL=https://staging.karakteristix.com/backend
SECRET_COOKIE_PASSWORD=YOUR_COOKIES_PASSWORD_32_CHARACTERS
```
